Astronomy Repository Resources Project

Overview

This project showcases my passion for astronomy while demonstrating my coding skills. The website features a glossary of astronomy terms with related YouTube videos and pageviews data.

Project Goals

  • Purpose: To share my interest in astronomy and provide educational resources. My aim is for individuals to explore more obscure astronomy concepts and be able to see how the popularity of terms changes over time or remains the same.
  • Target Audience: Amateurs and enthusiasts of astronomy.
  • Technologies Used: HTML, CSS, JavaScript, Node.js, Three.js, Web Scraping, APIs, Caching.

Filtering Description

Depending on the month of the year 2023 you select, you can also filter the dropdown list of terms based on their popularity (how often users visit the page for that term). There are four options you can choose from:

  • Most popular only: Terms with pageviews in the top 75th percentile.
  • Least popular only: Terms with pageviews in the bottom 25th percentile.
  • Somewhat popular: Terms with pageviews in between the 25th and 75th percentile.
  • Does not matter: All terms all displayed.

Key Milestones and Issues

Development Challenges

  • Animation Issues: Fixed twinkling stars background conflicting with the search bar.
  • API Rate Limits: Overcame by caching data and handling API errors.
  • Fetching Logic: Improved term fetching logic and handled edge cases.
  • Monthly Pageviews: Adjusted logic to fetch and display accurate data.

Deployment Hurdles

  • Encountered memory and CORS policy issues.
  • Current solution involves splitting the deployment between Netlify and Render.

Key Technologies and Frameworks

  • HTML, CSS, JavaScript: Core technologies for building the website.
  • Three.js: Front-end animations
  • Node.js: Backend development and server-side scripting.
  • Web Scraping: To gather astronomy terms and YouTube video data.
  • APIs: Wikipedia API for glossary terms, XTools for pageviews data, YouTube API for videos.
  • Caching: Implemented to store API responses locally and reduce redundant requests.

Final Thoughts

This project taught me a lot about web development, handling APIs, and debugging. I'm proud of the progress and the final product, which not only showcases my interest in astronomy but also my ability to overcome technical challenges.
